# Irregular-Memories-Middle
 如你所见，这是一个存放Irregular**初中全部照片**的仓库<br>
 不知道怎么说这种感觉，但是俗话说得好，**人不能同时拥有青春和对青春的感受**<br>
 时至今日我仍觉得初中是我整个学生阶段最开心的日子。大家都挺单纯的，没什么奇奇怪怪的逼事，学业不能说很简单，但是如今我看来并不难<br>
 当我真正走出初中的时候，才意识到曾经的美好。但是，**相见不如怀念**。<br>
 这就是我建立这个仓库最大的意义了：我希望把我这段可能是人生中最美好的回忆放在云端<br>
 这样不论走到哪里，我都可以想起那段时光，想起在三贤九义校度过的一切<br>
