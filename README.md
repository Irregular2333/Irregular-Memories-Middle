## 最后
最后就来简简单单说下做这个仓库的一些想法吧。<br>
人是不能同时拥有青春和对青春的感受的。在大学待了没到一年真正开始怀念从前，开始愿意相信，不是不怀念，只是没有到年纪罢了。<br>
最初这个仓库的想法还是直接用Github的直链来实现......这样就对梯子要求可能有点高了，在[**@pai233**](https://github.com/pai233)的提议和帮助下，最终选择了使用[**CloudFare**](https://www.cloudflare.com/)来对仓库中的图片进行中转，这样就可以做到不需要梯子就可以查看图片了，速度也比直接访问Github直链要快一点<br>
我也不知道说啥了，就这样吧。
